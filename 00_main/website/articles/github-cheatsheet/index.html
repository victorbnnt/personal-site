Initialize local repository

mkdir my-local-repository
cd my-local-repository
git init

Link local repository to remote

Note that this method requires SSH authentication to be set. See this good article.
Note that <local_name> used later should not already exist

  • Go to the remote repository in github.com
  • Click <>Code button
  • In the Local tab / SSH
  • Copy the url starting with git@
  • In the terminal, navigate to the local repository to be linked to the remote repository
  • Write the code
git remote add <local_name> <repository url>.git

Replace local link to remote repository

Note that this method requires SSH authentication to be set. See this good article.
Note that <local_name> used later should already exist

  • Go to the remote repository in github.com
  • Click <>Code button
  • In the Local tab / SSH
  • Copy the url starting with git@
  • In the terminal, navigate to the local repository to be linked to the remote repository
  • Write the code
git remote set-url <local_name> <repository url>.git

Show links to remote repository

  • In the terminal, navigate to the local repository
  • Write the code
git remote -v

Delete local repository link to GitHub

  • In the terminal, navigate to the local repository
  • Write the code
rm -rf .git

Basic commands

  • In the terminal, navigate to the local repository
  • Write the code
gh repo create # to create a GitHub repository
git status # displays the state of the working directory and the staging area
git add --all # adds a change in the working directory to the staging area. It tells Git that you want to include updates to all files in the next commit
git add <filename> # adds a change in the working directory to the staging area. It tells Git that you want to include updates to <filename> in the next commit
git commit -m '<message>' # captures a snapshot of the project's currently staged changes
git log # shows logs
git diff # shows differences
git branch -d <branch-name> # soft delete branch (will not work if everything as not been pushed)
git branch -D <branch-name> # hard delete branch
